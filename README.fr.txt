Cette application est spécialement développée pour mon Ecovacs Deebot T9+, mais peut également fonctionner avec d'autres Deebots avec fonction serpillière (les appareils de type 950).
L'application utilise la bibliothèque ecovacs-deebot.js développée par mrbungle64, et est dérivée de l'application originale Ecovacs pour Homey développée par Olivier Messiant
